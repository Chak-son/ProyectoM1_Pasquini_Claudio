# 🎨 Colorfly Studio®

**Colorfly Studio®** es una herramienta web moderna y minimalista para la generación de paletas de colores. Diseñada con un enfoque en la pureza del código y la experiencia de usuario.

## ✨ Características Principales

* **Generación Inteligente:** Crea paletas de 6, 8 o 9 colores con un solo clic.
* **Alineación Dinámica:** Organización visual inteligente (máximo 6 elementos por fila).
* **Código Semántico:** Desarrollado 100% sin etiquetas `<div>`.
* **Gestión de Colores:** Copiado rápido de códigos **HEX/HSL** y sistema de bloqueo.

## 🚀 Tecnologías Utilizadas

* **HTML5 Semántico**
* **CSS3 (Flexbox & Animations)**
* **JavaScript ES6+**
